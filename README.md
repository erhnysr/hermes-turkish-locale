# 🇹🇷 Turkish Locale Skill Pack for Hermes Agent

A complete Turkish locale ecosystem for Hermes Agent — real-time crypto prices in TRY, Turkish news aggregation, styled PNG briefing cards, and Telegram cron automation.

## What's Inside

- **turkish-news** — Hürriyet, Bloomberg HT, NTV, CoinTelegraph TR
- **bist100** — BIST100, USD/TRY, EUR/TRY, Gram Altın
- **turkish-daily-brief** — Automated daily briefing via Telegram
- **scripts/** — CoinGecko API, 1200px PNG card generator, Telegram delivery

## Install
```
hermes skills install github.com/erhnysr/hermes-turkish-locale
```

## Requirements
```
pip install pillow requests
```

## Zero API Keys Required

Built entirely with free public APIs (CoinGecko, RSS feeds).

Built for the Hermes Agent Hackathon 2026 🚀
